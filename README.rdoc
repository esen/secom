== SECOM

SECOM is an educational institution which teaches pupils languages, computer skills,
prepares for TOEFL, ORT tests. It has buildings in 5 places of Kyrgyzstan. Currently,
they are using papers and Microsoft Excell for all needs of course management. This
application can be used remotely from everywhere. The application is for managing the
course lists, schedules, student lists, as well as for making reports like “karne”,
teacher engagement, course statistics.
